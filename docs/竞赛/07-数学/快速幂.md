## 模板

```cpp
ll Pow(ll a,ll b)
{
	a%=mod;
	ll res=1;
	while(b)
	{
		if(b&1)res=res*a%mod;
		a=a*a%mod;
		b>>=1;
	}
	return res;
}
```

## 例题

:::info[[洛谷 P1226 【模板】快速幂](https://www.luogu.com.cn/problem/P1226)]

给定三个整数 $a,b,p$，求 $a^b \bmod p$。（$a,b,p\le2^{31}$）

:::

```cpp
#include <bits/stdc++.h>
using namespace std;

using ll=long long;
int mod;
ll Pow(ll a,ll b)
{
	a%=mod;
	ll res=1;
	while(b)
	{
		if(b&1)res=res*a%mod;
		a=a*a%mod;
		b>>=1;
	}
	return res;
}
int main()
{
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	ll a,b;
	cin>>a>>b>>mod;
	cout<<a<<'^'<<b<<" mod "<<mod<<'='<<Pow(a,b)<<'\n';
	return 0;
}
```
