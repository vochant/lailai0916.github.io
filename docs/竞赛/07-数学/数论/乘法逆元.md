# 乘法逆元

## 费马小定理

$$
\begin{align*}
  & ii^{-1}\equiv 1 \pmod p \\
  & ii^{-1}\equiv i^{p-1} \pmod p \\
  & i^{-1}\equiv i^{p-2} \pmod p
\end{align*}
$$

```cpp
#include <bits/stdc++.h>
using namespace std;

const int mod=998244353;
using ll=long long;
ll Pow(ll a,ll b)
{
	a%=mod;
	ll res=1;
	while(b)
	{
		if(b&1)res=res*a%mod;
		a=a*a%mod;
		b>>=1;
	}
	return res;
}
int main()
{
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	ll n;
	cin>>n;
	cout<<Pow(n,mod-2)<<'\n';
	return 0;
}
```

## 线性逆元

$$
\begin{align*}
  & p\bmod i=p-\left\lfloor\dfrac{p}{i}\right\rfloor\cdot i \\
  & p\bmod i=-\left\lfloor\dfrac{p}{i}\right\rfloor\cdot i \\
  & i^{-1}\cdot(p\bmod i)=-\left\lfloor\dfrac{p}{i}\right\rfloor \\
  & i^{-1}=-\left\lfloor\dfrac{p}{i}\right\rfloor(p\bmod i)^{-1}
\end{align*}
$$

```cpp
ll inv[N];
void init()
{
	inv[0]=inv[1]=1;
	for(int i=2;i<N;i++)
	{
		inv[i]=(mod-mod/i)*inv[mod%i]%mod;
	}
}
```
