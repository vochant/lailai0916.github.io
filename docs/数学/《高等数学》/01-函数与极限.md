# 函数与极限

| 考点 | 重要程度 |   分值    |     常见题型     |
| :--: | :------: | :-------: | :--------------: |
| 函数 |    ★★    | $0\sim3$  |    选择、填空    |
| 极限 |   必考   | $6\sim10$ | 选择、填空、大题 |

## 函数

### 定义域

1. 分母不为 $0$：$\dfrac{1}{f(x)}\Rightarrow f(x)\not=0$
2. 偶次根式：$\sqrt[2n]{f(x)}\Rightarrow f(x)\ge0$
3. 对数函数：$\ln{f(x)}\Rightarrow f(x)>0$
4. 反三角函数：$\arcsin{f(x)},\arccos{f(x)}\Rightarrow -1\le f(x)\le 1$
